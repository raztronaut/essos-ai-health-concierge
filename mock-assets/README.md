# Essos Mock Asset Pack

This folder is the source-of-truth fixture pack for the Essos AI Concierge work
trial. It supports two uses:

- polished PDFs for the demo (`mock-assets/pdf/essos/`)
- structured data for the Convex seed (`scripts/seed.ts` -> `convex/seed.ts`)

All patient names, handles, clinics, hotels, drivers, confirmation numbers, and
records are fictional. The only content derived from the provided project
context is the Maya rhinoplasty itinerary/pre-op source material from
`.context/example_itineraries.md`.

## Structure

- `patients/*.json` holds patient profiles, itineraries, care-instruction
  chunks, seeded conversations, and source-document links.
- `source-docs/*.md` holds the human-readable source documents that generate
  PDFs and `source_documents` rows.
- `manifest.json` is generated by `scripts/generate_pdfs.py`.
- `scripts/generate_pdfs.py` writes PDFs and preview PNGs under
  `mock-assets/pdf/essos/`.

## Source Statuses

- `verified`: based on project context or an Essos operating note in the demo.
- `demo_notional`: realistic mock content for the work trial, not a clinic
  source of truth.
- `personalized_pending`: deliberately incomplete because the clinic provides
  patient-specific guidance later.
- `missing`: not captured in the demo system.

## Agent Semantics

Eve may answer from `answer_reference` docs when the patient's question is
routine and directly supported by the source. Eve must escalate when a document
is `escalate_only`, when the source is missing or personalized pending, or when
the question involves medication decisions, post-op recovery judgment, symptoms,
staff safety, or out-of-package changes.

## Generate PDFs

```bash
pnpm assets:generate                          # canonical entry point (Node wrapper -> Python)
python3 mock-assets/scripts/generate_pdfs.py  # or invoke the generator directly
```

The generator uses ReportLab and Poppler and regenerates the committed PDFs and
`manifest.json` (with per-doc `sha256`). In Codex, prefer the bundled Python
runtime and Poppler if needed.
